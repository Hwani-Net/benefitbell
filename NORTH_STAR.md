# ⭐ 혜택알리미 북극성 (NORTH STAR) & 안티테제 전략

## 🎯 목표
기존 고래 앱들(보조금24, 캐시노트 등)의 치명적 약점을 정면 공략하여, 사용자가 **스스로 찾지 않아도 혜택이 먼저 찾아오고, 3초 만에 결과(해당됨/안됨)를 알 수 있는 유일한 앱**을 만든다.

---

## 💥 경쟁사 약점 (Pain Points)
1. **광고 범벅 알림**: 캐시노트 알림의 99%가 광고/대출 → 알림을 끄거나 앱 삭제.
2. **"나 해당돼?" 판단 불가**: 보조금24는 검색만 제공. 복잡한 관공서 문서를 읽고 직접 판단해야 함.
3. **정보 과부하**: 앱을 열면 나와 상관없는 혜택 수백 개가 쏟아짐.
4. **2앱 핑퐁**: 공고는 정부24, 자격은 복지로 → 원스톱 신청 불가.

---

## ⚡ 10x 킬러 피처 (구현 우선순위)

### 🥇 1. "나 해당돼?" 3초 AI 자격 판정 (최우선 구현)
- **단계별 프로필 입력**: 첫 진입 시 나이/직종 등 최소 정보만 입력. 거부감 최소화.
- **정확도 향상 유도**: "더 자세히 입력하면 혜택 정확도가 올라갑니다" 안내로 추가 정보 입력 유도.
- **AI 필터링**: 100개 혜택 → 내 혜택 20개로 압축.
- **% 배지**: 내 프로필 기반으로 **수령 가능성(%)** 즉시 표시. 직접 읽고 판단할 필요 없음.

### 🥈 2. "돈 되는 알림" (광고 0% 맞춤 푸시)
- **광고 제로**: 내 프로필에 맞는 **진짜 혜택**만 알림.
- **재알림 금지**: 한 번 발송된 새 혜택은 시스템이 오지랖 부려 다시 보내지 않음 (스팸 방지).
- **능동적 리마인더**: 사용자가 직접 "나중에 다시 알림"을 체크한 항목만 재알림.
- **안 읽음 뱃지**: 앱 내 UI로만 안 읽은 혜택을 조용히 표시.

### 🥉 3. 혜택 원스톱 (클린 프리미엄 UX)
- 공고 내용 + AI 3줄 요약 + "해당됨" 결과 + **신청 링크**를 한 화면에 통합. 무관한 혜택은 자동 숨김.
- **기간 자동 필터링**: 모집 마감된 혜택은 매일 갱신되는 스크립트를 통해 자동으로 삭제/숨김 처리. "들어갔더니 끝남" 경험 원천 차단.
- **필요 서류 다이렉트 링크 (서류 핑퐁 방지)**: 공고만 텍스트로 주는 게 아니라, 무슨 서류가 필요하고 **어디서 온라인 발급받는지 (정부24 등) 바로가기 링크**를 매칭해서 제공 (추후 API 연동을 통한 대리 발급 기능 확장 검토).
