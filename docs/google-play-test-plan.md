## 4. 테스트 시나리오 (핵심 사용자 여정)

### TC-01: 온보딩 플로우
```
앱 설치 → 첫 화면 로드 → 카카오 로그인 → 프로필 위저드 (3단계)
→ 기본정보 입력 → 추가정보 입력 → 사업자정보(선택) → 홈 화면
```
- **검증**: 위저드 단계별 진행, 필드 저장, 뒤로가기 처리
- **기대 결과**: 3단계 완료 후 맞춤 추천 표시

### TC-02: 맞춤 혜택 추천
```
홈 화면 → 맞춤 추천 섹션 → 수령 가능성 % 배지 확인
→ 혜택 상세 → AI 자격 판정 → 3줄 요약 확인
```
- **검증**: 프로필 기반 매칭 정확도, AI 응답 속도
- **기대 결과**: 프로필에 맞는 혜택 상위 노출, AI 분석 < 5초

### TC-03: 검색 및 필터
```
검색 아이콘 → 키워드 검색 → 결과 리스트 → 카테고리 필터
→ 상세 페이지 → 서류 체크리스트
```
- **검증**: 검색 정확도, 필터 동작, 빈 결과 처리
- **기대 결과**: 관련 혜택 즉시 표시, 필터 조합 정상

### TC-04: 서류 체크리스트 & 원스톱
```
혜택 상세 → 필요 서류 목록 → 체크박스 체크 → 정부24 발급 링크
→ 외부 링크 이동 → 돌아오기 → 체크 상태 유지
```
- **검증**: 체크 상태 localStorage 저장, 외부 링크 정상 작동
- **기대 결과**: 앱 재진입 시 체크 상태 유지

### TC-05: 다크모드 & 다국어
```
설정 → 다크모드 토글 → 전체 UI 확인
→ 영어 전환 → 주요 화면 확인 → 한국어 복귀
```
- **검증**: 색상 대비, 텍스트 가독성, 번역 누락
- **기대 결과**: WCAG AA 대비비 충족, 영어 전환 정상

### TC-06: 푸시 알림
```
알림 권한 허용 → FCM 토큰 등록 → 알림 수신
→ 알림 탭 → BottomNav 배지 → 읽음 처리
```
- **검증**: 토큰 등록, 알림 수신, 배지 카운트
- **기대 결과**: 알림 정상 수신 + 배지 업데이트
